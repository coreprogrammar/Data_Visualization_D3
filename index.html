📰 Chart Overviews

📊 Bar Chart

Displays the total goals scored by each team. Users can filter by match type or group to analyze offensive performance.

🥧 Pie Chart

Visualizes how goals are distributed among regions or teams. Ideal for comparing overall team contributions.

🔀 Sankey Diagram

Shows the progression of teams through tournament stages and goal flow using connected node links.

🟢 Circle Pack

Teams are represented as circles sized by total goals. Great for visual comparison of scoring magnitude.

📈 Line Chart

Plots goals scored over time. Helps users track match activity and identify scoring spikes across dates.

🌳 Tree Map

Organizes teams by group and goal count. Each block’s area corresponds to the number of goals scored.

🕸️ Radar Chart

Compares multiple team performance metrics like goals, wins, losses in a radial format. Useful for spotting strengths and weaknesses.

📶 Stacked Bar Chart

Breaks down total goals by each stage of the tournament per team. Highlights where each team performed best.

⚽ Bubble Chart

Plots total goals vs matches played. Bubble size represents wins, offering multi-dimensional comparison in a single glance.

🍩 Donut Chart

Displays win, draw, and loss distribution in a donut layout. Each slice is color-coded and labeled with percentage values.

📉 Goal Difference Chart

Horizontal bar chart sorted by goal difference (scored - conceded). Highlights top-performing defensive/offensive teams.

🔥 Heatmap

Shows daily goal activity using color intensity. Helps identify high-scoring match days and scoring trends over the tournament.
